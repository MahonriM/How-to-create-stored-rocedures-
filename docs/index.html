Introduction

Welcome to this blog in which we are going to learn how to create stored procedures. In the word of databases we have several tools for stored data. With a DBMS we can create a lot of things such as databases,tables, functions,stored procedures.


What is a stored procedure?

It is a prepared SQL code that we can use for read data, insert data,delete data and update data. If you use a lot of DML commands, you should use stored procedures.In some cases is better to use stored procedures than the DML command. Sometimes we can implement stored procedures when we are doing a CRUD application.

How to make a stored procedure?

We can make stored procedure in several ways in this blog we are going to show how to make stored procedures in SQL Server and MySQL. This two DBMS are pretty similar there are a little different in the part of stored procedures. SQL Server use the Transct-SQL and MySQL use SQL.SQL Server is a product of Microsoft while MySQL is from Oracle.

In SQL Server we can create a stored procedure following the next scructure

  • In this procedure we start with the word create and then procedure or proc are the same then we have to put a name to our procedure. If our procedure use a parameter we can declare as many as we need with @name type separated by a , each parameter. The third part is use the words as and begin. Finally we can use the DML command in this case insert and the word end
  • Create procedure nameofprocedure
  • @parmaters type
  • as begin
  • Insert into tablename values(@parmaters)
  • END

Example of procedure in SQL Server

Create procedure spaddcamion @idcamion int, @descripcion varchar(50), @placa varchar(20),@marca varchar(30),@modelo varchar(30) as Begin Insert into camion values(@idcamion,@descripcion,@placa,@marca,@modelo) END
